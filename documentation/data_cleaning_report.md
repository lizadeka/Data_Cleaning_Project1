# SQL Data Cleaning Project - Cleaning Project 1

## Objective
The objective of this project is to clean a messy orders dataset using SQL

## Data Issues Identified -
- Duplicate Records
- Null emails
- Invalid email formats
- Extra spaces in customer names
- City spelling mistakes
- Multiple date formats
- Missing dates
- Negative order amounts

## Cleaning Steps -
Query 1 - Detect duplicate records
Query 2 - Remove duplicate recods
Query 3 - Find null emails
Query 4 - Replace missing email
Query 5 - Detect invalid emails
Query 6 - Fix invalid emails
Query 7 - Fix extra spaces in customer_name
Query 8 - Fix city spelling mistakes
Query 9 - Remove extra space in city column
Query 10-12 - Standardize date formats
Query 13-14 - Handle missing dates
Query 15 - Fix negative amounts
Query 16 - Replace null amounts

## Final Step -
Coverted incorrect datatypes to correct SQL datatypes
